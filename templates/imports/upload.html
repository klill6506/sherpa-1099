Expected File Format

Your file should contain columns for recipient information and payment amounts. Column names will be auto-mapped to the following fields:

Recipient Info

  • name - Recipient name
  • tin or ssn - Tax ID
  • address - Street address
  • city, state, zip

1099-NEC

  • amount or compensation - Box 1
  • federal withheld - Box 4

1099-MISC

  • rents - Box 1
  • royalties - Box 2
  • other income - Box 3
  • medical - Box 6
